# الدليل التقني ودليل النشر - AromaVerse
**المشروع:** هندسة Perfume Nexus
**الاستضافة:** بنية تحتية سحابية خاصة - سيدي عبد الله (الجزائر)

## 1. هندسة النظام
تعتمد المنصة على تقنيات حديثة وآمنة:
- **الواجهة الأمامية (Frontend):** React 18 / Vite / Tailwind CSS (محسن للغة العربية والفرنسية).
- **الواجهة الخلفية (Backend):** API Serverless محسنة لمراكز بيانات سيدي عبد الله.
- **قاعدة البيانات:** PostgreSQL (الإصدار 16+) مع دعم JSONB لمرونة تركيبات العطور.

## 2. إعدادات النطاق و DNS
لربط اسم النطاق الخاص بك بالخوادم المحلية، قم بضبط السجلات التالية لدى مزود الخدمة (Hostinger/Namecheap):

| النوع | المضيف | القيمة | ملاحظة |
| :--- | :--- | :--- | :--- |
| **A** | @ | `105.XXX.XX.XX` | عنوان IP الخاص بمركز بيانات سيدي عبد الله |
| **CNAME** | www | `aroma-verse.dz` | توجيه إلى النطاق الرئيسي |
| **TXT** | @ | `v=spf1 include:_spf.google.com ~all` | إعدادات البريد الإلكتروني |

## 3. الأمان وشهادة SSL
- تفرض المنصة الاتصال عبر **HTTPS (TLS 1.3)**.
- يتم إدارة شهادة SSL عبر **Let's Encrypt** (تجديد تلقائي كل 90 يومًا).

## 4. الصيانة والنسخ الاحتياطي
- **موقع النسخ الاحتياطي:** خادم تخزين إضافي في سيدي عبد الله.
- **التكرار:** يوميًا (الساعة 02:00 صباحًا).
- **مدة الاحتفاظ:** 30 يومًا.
